# Site do Edital LTD 2026.1

Projeto estático pronto para publicação da landing page institucional do `EDITAL Nº 01/2026.1`.

## Estrutura

- `index.html`: estrutura principal da página.
- `assets/css/styles.css`: identidade visual, layout responsivo e animações.
- `assets/js/content.js`: textos, datas, vagas, responsáveis e link de inscrição.
- `assets/js/app.js`: renderização dos blocos, menu mobile e animações de entrada.
- `assets/img/logo-estacio.jpeg`: logo utilizada no topo.
- `assets/docs/edital-01-2026-1-publicacao.pdf`: PDF oficial para download.
- `assets/favicon.png`: favicon do site.

## Edição rápida

1. Ajuste textos, datas, vagas, responsáveis e link de inscrição em `assets/js/content.js`.
2. Se quiser trocar a logo, substitua `assets/img/logo-estacio.jpeg`.
3. Se quiser trocar o favicon, substitua `assets/favicon.png`.

## Publicação

Abra `index.html` no navegador ou publique toda a pasta em qualquer hospedagem estática.
